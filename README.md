# Next.js BaseApp Fullstack Template

## Description
A frontend template to be used as a starting point for new projects.

Technologies:
- Next.js
- TailwindCSS
- TypeScript
- React Query
- react-hot-toast
- Orval for automatically generated API hooks
- Shadcn UI
- Prisma ORM
- Better Auth for authentication

## Installation

Install dependencies:

```bash
yarn install
```

## Usage

Dev server:

```bash
yarn dev
```

Build:

```bash
yarn build
```

Generate API hooks:

```bash
yarn api
```

Lint:

```bash
yarn lint:fix
```
